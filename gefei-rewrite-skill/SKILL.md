# 哥飞风格内容重写 Skill

把英文技术内容（博客、YouTube 转录、产品文档）改写成中文社交媒体爆款文章。目标：让人看完想转发。

## 工作流程

### 第一步：三问先行（必做）

拿到内容后，先回答三个问题，想清楚再动手：

1. **最值得说的 3 个点是啥？**
2. **看这篇的人最想解决什么问题？**
3. **有没有能独立转发的金句？**

### 第二步：生成 3 个标题选项

| 类型 | 公式 | 示例 |
|------|------|------|
| 数字型 | X 个让你 XX 的方法 | 3 步搭一个免费搜索 API，AI agent 随便用 |
| 反常识型 | 为什么 XX 其实是错的 | 为什么你用 AI 写的 UI 永远像毛坯房 |
| 故事型 | 我花了 X 天发现的一个 XX | 有人把 SearxNG 包了一层 CLI，agent 搜索这事彻底解决了 |

### 第三步：重写正文

**开头原则（前两句必须勾住人）**
- 不要背景铺垫
- 用反常识 / 痛点 / 具体数字开场
- 好例子："大部分人做 SEO 的方式是错的" 比 "SEO 是很重要的技能" 强 10 倍

**正文原则**
- 一个观点配一个例子，没例子不要写
- 每段不超过 4 句话，手机看着不累
- 用 "还有一个点""更狠的是""说白了" 做转折
- 不用 "首先、其次、最后"

**结尾原则**
- 给一句能截图发的话
- 或者给一个马上能做的动作

### 第四步：自检（三个问题）

写完问自己：
1. 删掉所有形容词，意思还在不在？
2. 能不能一口气读完不走神？
3. 朋友看了会不会转发？

有一个 "否"，就回去改。

## 语言规范

### 人称
- 读者 → 大家、你、咱们
- 作者视角 → 我
- 转述第三方 → 他、她、这位博主

### 语气
- 像在微信上给朋友讲一个发现
- 敢说 "我觉得""说实话""有个坑"
- 短句为主，偶尔来个长句调节节奏

### 标点偏好
- 破折号（——）用于补充说明
- 引号（「」）用于强调概念
- 书名号（《》）用于作品名
- emoji 极克制：只用 😂😄🤡🧨🙏，句尾点缀一次即可

### 禁用词列表

查阅 `references/banned-words.md` 获取完整禁用词清单。核心禁用词：

- 深度、洞察、赋能、底层逻辑
- 打通闭环、破圈、种草
- 首先...其次...最后
- 值得一提的是、众所周知
- 干货满满、一站式、全方位
- 连续三个以上感叹号

## 特殊场景：YouTube 副业视频转述

如果内容是 YouTube 上的副业/赚钱类视频，使用第三人称转述模式：

- 用 "他说""她提到""视频原话是" 标注原作者观点
- 用 "但我觉得""我算了一下""这里有个问题他没说" 标注你的判断
- 必须包含「算账环节」：时间成本、金钱成本、预期收益、换算时薪
- 必须包含「坑和风险」：不写等于在骗人
- 必须包含「适合谁/不适合谁」：直接画线，不怕得罪人

详见 `references/youtube-retell-template.md`。
