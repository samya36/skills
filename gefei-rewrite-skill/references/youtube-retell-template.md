# YouTube 副业视频转述模板

当内容来源是 YouTube 上的副业/赚钱类视频时，使用此模板。

## 核心定位

你是「帮大家筛选过、算过账、加过判断」的信息中介：
- 原视频的信息如实转述，但会标注哪些是「他说的」哪些是「我的判断」
- 对原视频的夸大、模糊、遗漏之处，你会补充或质疑

## 人称规范

| 角色 | 用语 |
|------|------|
| 原作者 | 他、她、这位博主、视频里的哥们/姐们 |
| 读者 | 你、大家 |
| 你自己 | 我（用于加判断、加补充时）|

## 信源标注

| 场景 | 用语示例 |
|------|----------|
| 转述原作者观点 | "他提到""她说""视频里讲的是""原话大概是" |
| 加入你的判断 | "但我觉得""我算了一下""这里有个问题他没说""我补充一点" |
| 不确定的信息 | "他没细说""我估的""待验证" |

## 标题公式

1. 这个老外靠 XX 月入 $X 万，我帮你拆了一遍
2. YouTube 上这个副业视频火了，我看完替你算了笔账
3. 有个博主说靠 XX 赚了 X 万，我来说说哪些能信
4. 我看了 XX 的赚钱视频，帮你划个重点、挑个刺
5. 这个副业思路我在 YouTube 上看到的，转述 + 我的判断

## 开头模板（60–120 字）

```
大家好，今天转述一个我在 YouTube 上看到的案例。

原视频是 [博主名/频道名]（链接放评论区）。

他讲的是 [一句话概括主题]。

我看完之后觉得：[你的第一反应/整体判断]——下面展开说。
```

## 正文结构

### 1. 原视频概要（如实转述）

必须覆盖：
- 这位博主是谁？什么背景？（如果视频里提了）
- 他做的是什么副业/赚钱方式？
- 他声称的收入是多少？周期多长？
- 他讲的核心方法/步骤是什么？

全程用「他说」「他的做法是」明确标注。

### 2. 算账环节（你来算）

**必须包含这张表：**

| 项目 | 数据 | 备注 |
|------|------|------|
| 时间成本 | XX 小时/天 or 周 | |
| 金钱成本 | XX 美元/人民币 | |
| 预期收益 | XX/月 or 单 | |
| 换算时薪 | 大约 XX | |
| 回本周期 | XX 天/月 | |

如果数据模糊，标注「这个数字他没细说，我估的」。

### 3. 我的判断/补充/质疑

必须覆盖：
- 他没提到的坑/成本/前提条件
- 这个方法的适用边界（什么人能复制、什么人别试）
- 有没有幸存者偏差/时间窗口已过/地域限制
- 我觉得可信的部分 vs 存疑的部分

### 4. 适合谁/不适合谁

直接画线，不怕得罪人：

```
✅ 适合：[具体人群画像]
❌ 不适合：[直接说]
```

### 5. 如果你想试，第一步做什么？

给一个最小可执行的起手式。
不是「先去学习」这种废话，是「先做 XX 验证 XX」。

## 结尾模板（40–80 字）

```
好了，以上就是这个视频的转述 + 我的判断。

原视频链接：[放这里 or 评论区]

我的观点不一定对，你自己判断——毕竟钱和时间是你的。

有问题可以留言，能回的我会回 🙏
```

## 绝不做的事

- 不无脑吹原视频（该质疑就质疑）
- 不无脑黑（人家确实有干货就承认）
- 不隐藏「他说的」和「我说的」的区别——必须标清楚
- 不编数据（原视频没说的就写「他没提」「我估的」）
- 不用「轻松」「躺赚」「零成本」等诱导词
- 不把「个例」包装成「人人可复制」
