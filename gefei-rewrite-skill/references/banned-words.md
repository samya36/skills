# 禁用词清单

改写时**绝对不能出现**的词汇和表达方式。

## 营销黑话

| 禁用词 | 为什么禁 |
|--------|----------|
| 深度 | 空洞、无信息量 |
| 洞察 | 同上 |
| 赋能 | 典型 AI 味/企业味 |
| 底层逻辑 | 装逼词 |
| 打通闭环 | 同上 |
| 破圈 | 营销黑话 |
| 种草 | 小红书黑话，但用烂了 |
| 干货满满 | 自吹 |
| 一站式 | 广告词 |
| 全方位 | 同上 |
| 颗粒度 | 企业黑话 |
| 抓手 | 同上 |
| 闭环 | 同上 |
| 对齐 | 同上 |
| 拉通 | 同上 |
| 沉淀 | 同上 |
| 复盘（慎用） | 可以用，但别滥用 |

## 过渡词/套话

| 禁用表达 | 替代方案 |
|----------|----------|
| 首先...其次...最后 | 还有一个点 / 更狠的是 / 说白了 |
| 值得一提的是 | 直接说 |
| 众所周知 | 删掉，直接说事实 |
| 不言而喻 | 同上 |
| 毋庸置疑 | 同上 |
| 综上所述 | 删掉 |
| 总而言之 | 删掉 |

## 标点禁忌

| 禁用 | 说明 |
|------|------|
| ！！！ | 最多两个感叹号，通常一个就够 |
| 「」滥用 | 不是什么都要加引号 |
| ... 滥用 | 省略号偶尔用，别每段都用 |

## AI 味表达

这些表达一看就是 AI 写的，必须改掉：

- "在当今...的时代"
- "随着...的发展"
- "让我们一起来看看"
- "接下来，我将为你介绍"
- "希望对你有所帮助"
- "如果你觉得有用，欢迎点赞收藏"
- "感谢你的阅读"
- "本文将从以下几个方面"
- "下面我们来详细了解一下"

## 副业/赚钱类特别禁用

| 禁用词 | 为什么禁 |
|--------|----------|
| 轻松月入 X 万 | 诱导词 |
| 躺赚 | 同上 |
| 零成本 | 骗人 |
| 睡后收入 | 滥用 |
| 被动收入（慎用） | 可以用，但要有实际支撑 |
| 财务自由 | 滥用 |
| 副业刚需 | 滥用 |
